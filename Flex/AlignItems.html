• Align-items --> cross axis

  • Align the "Individual items" based on the Flex direction

  • Align the "Next Starting items" based on the Flex direction

  • The cross axis is the axis perpendicular to the main axis inside a flex container.

  • It is perpendicular to the main axis. If the main axis is horizontal, the cross axis is vertical.

  • If "flex-direction:row" is used Then it will Aligns "Next Starting items" based on Rows.

  • If "flex-direction:column" is used Then it will Aligns "Next Starting items" based on Columns.
